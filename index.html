Source

Source is an open-source dictionary/almanac for developers. In the process of my own learning I have often found myself wondering what the meaning of a specific term in a current context is. Source allows you to search for vocabulary terms with context and returns a formal meaning and a layman's meaning. In addition to the definitions, Source links related open-source GitHub repositories and tutorial sites for the concept related to the term.
